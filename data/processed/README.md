# Course Data Extraction

This dataset is generated by extracting structured information from the official **IISER Pune Course Content PDFs**.

The extraction script reads course PDF files and converts them into a structured **JSON format** that can be used for:

- Curriculum graph modeling
- Course dependency analysis
- Degree pathway planning
- Dashboard visualization

Each PDF corresponds to **one course** and is parsed to extract key metadata.

---

# Output File

The script generates:


This file contains a **list of course objects**.

---

# JSON Structure

Each course is represented as a JSON object with the following fields.

| Field | Type | Description |
|------|------|-------------|
| `course_code` | string | Unique identifier for the course |
| `title` | string | Official course title |
| `semester` | number | Semester for which the course is open |
| `nature` | string | Course format (Lecture, Lab, Tutorial, Project, etc.) |
| `credits` | number | Number of credits assigned to the course |
| `prerequisites` | string | Required background or prerequisite courses |
| `objectives` | string | Description of course goals and learning objectives |
| `course_content` | string | Detailed syllabus or topics covered |
| `evaluation` | string | Assessment structure (mid-sem, end-sem, quizzes etc.) |
| `readings` | string | Recommended textbooks or references |
| `file` | string | Source PDF filename |

# Notes

Each JSON entry corresponds to one course PDF.

Some fields may be empty if the information is not available in the source document.

Prerequisites are currently stored as raw text and may require additional parsing for dependency graph construction.

Use in Curriculum Graph

This dataset will be used to construct a directed course dependency graph where:

Nodes represent courses

Edges represent prerequisite relationships

Example:


Real Analysis I → Complex Analysis

This structure enables:

prerequisite visualization

course ordering

pathway analysis for majors and minors
